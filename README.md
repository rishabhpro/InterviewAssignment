# React Interview Assignment

This is a React interview assignment project that demonstrates my skills in building a TypeScript React application. The project fetches data from a product API, displays the products in visually appealing cards, and provides functionality to manipulate the quantity and delete products.

## Project Setup

To set up the project locally, follow these steps:

1. Clone this repository to your local machine.
2. Install the dependencies by running the following command: `npm install`.
3. Start the development server with the command: `npm run dev`.

## Project Requirements

- Create a TypeScript React application using either `create-react-app` or `Vite`.
- Create a GitHub repository to submit your assignment for review.
- Fetch product data from the [Product API](https://dummyjson.com/products).
- Display the products in cards with the following information:
  - Product image
  - Title
  - Price
  - Discount
  - Quantity
- Ensure proper alignment of all product card content.
- Use a layout system, such as a grid, to display the product cards.
- Design the layout to be responsive on small mobile devices.
- Provide options to increase and decrease the quantity of each product.
- Include an option to delete the product.
- Commit your changes to the Git repository with meaningful commit messages.

## Libraries Used

The following libraries were used in this project:

- React: JavaScript library for building user interfaces.
- Axios: Promise-based HTTP client for making API requests.
- React Icons: Icon library for React components.
